Transponder to IF Formula Sheet.

The information contained below is meant to help you convert, RF Frequencies to IF Frequencies. This is assuming that you are working on a Non-Inverted C band Service. For an Inverted C band or LBand Service. Please refer to the MTN_Freq   sheet linked to this page.



To Calc the Center Frequency of a Modem for a non inverting C-Band Service apply the following formula. 

IF Center = 140 + (TXCF -XPCF)

Example  JED.1
TX RF:                6033.0800 
Transponder Center:   6050.0000 
                      _________ 
Gives an IF FReq of    123.0800
 

To Calc the Center Frequency of a DeModulator for a non inverting C-Band Service apply the following formula. 

IF Center = 140 + (RXCF -XPCF)

Example  JED.1
RX RF:                3836.2500 
Transponder Center:   3825.0000 
                      _________ 
Gives an IF FReq of    151.2500
